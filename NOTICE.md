# Data notice

This repository publishes a derived evidence index, not the original source workbooks or database releases.

The scientific records remain attributable to their original publications and databases. Links to authoritative sources are provided on the website and in `data/datasets.json`. HGNC annotation data are distributed under CC0. GenAge and LongevityMap use remains subject to the Human Ageing Genomic Resources conditions. NCBI content is presented with links to NCBI's disclaimer and policies.

The consolidated workbook defines gene eligibility. Displayed evidence comes from the underlying public transcriptomic, epigenetic, LongevityMap, and GenAge source files. Records are linked to approved HGNC symbols only when identity mapping is unambiguous.

The atlas does not independently reproduce the original statistical models. Association evidence does not establish causality. Users should consult the source publication and full source data before drawing biological or clinical conclusions.

This release does not include a standalone proteomic dataset. Empty evidence categories are omitted from individual gene pages.
